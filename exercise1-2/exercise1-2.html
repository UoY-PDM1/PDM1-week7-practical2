Exercise 1.2

Your sketch:

Instructions

The assets folder contains two images. Load both into your sketch. Use blue-sky.jpg as the sketch background. Once it's loaded, use its width and height properties to set the size of the canvas. Create a flying planes animation using plane.png (see the video below). Your sketch should meet the following requirements:

  • The sky should fill the canvas.
  • There should be at least two aeroplanes travelling across the sky from left to right at different altitudes (different y locations).
  • One plane should appear further away (i.e. smaller) and should travel faster than the other.
  • Once a plane passes out of view on the right side of the canvas, reset its x position so it can travel across the sky again.

Expected output:

To create two planes, you only need to load plane.png once. You can call image() with the same loaded image more than once and set the displayed size to be different each time.

The plane image is large so you will want to display it much smaller e.g. at a quarter or a third of its actual size.

You will need two global variables to track the x position of each plane and separate conditionals to figure out if an individual plane has left the right side of the canvas.

You could consider writing a Plane class although this is not essential.
