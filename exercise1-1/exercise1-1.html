Exercise 1.1

Your sketch:

Instructions

Create a canvas that is 512 pixels by 410 pixels. Load the image in the assets folder and display it centred and scaled down so that the whole image fits on the canvas. The image should slowly zoom in until the height of the displayed image is the same height as the original image and the cat's face takes up most of the canvas.

Expected output:

To make it easier to centre the image, use imageMode(CENTER). It works the same way as rectMode(CENTER).

To scale the image to fit the canvas (rather than the other way round), use the optional width and height properties of the image() function.

You will need to display the image at the width and height of the canvas to start with, then increase the width and height of the displayed image one pixel at a time until it reaches the size of the original image. You can use the p5.Image width and height properties.
